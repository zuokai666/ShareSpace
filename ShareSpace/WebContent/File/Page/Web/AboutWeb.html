网站介绍

大家好，我是King，这是我做的社交网站，英文名字是Share Space，中文名字是分享空间， 我们的口号是分享空间，留住你我，旨在为大家提供一个交流的平台。 我从2016年9月24日下午开始建站，到现在10月10日下午为止完成了Share Space Beta 1.0版本。 目前网站功能较简单，包括用户的注册与登录，个人信息的修改，发表分享与评论，推荐和反对他人的分享， 查看用户主页，查看Rank分数排行，查看关系图等等功能，但是它为了我提供了一个很好的想法的实现平台， 也为大家提供了一个交流的平台。后续的一些功能将在Beta 1.1版本中实现，目前的构想是，加入：互粉功能， 举报他人分享，管理分享，完善用户的个人信息，网站公告，用户反馈，系统认证，收发短消息等等功能。

网站特色

Rank评算：

发表分享+2，如果删除分享，Rank将下降2分，但伙伴之间的关系分数不影响；
他人评论+1；
他人推荐+2；
他人反对-2；
加我关注+10；
取消关注-10；
他人发信+2；
举报非法分享（管理员按情况加分）；
提交Bug和建议（管理员按情况加分）；

伙伴之间的Relation评算（同理）：

他人评论+1；
他人推荐+2；
他人反对-2；

关于创新问题

很多人都在说，你这个网站也没啥好的吗，这不就是抄袭博客园，新浪微博等等嘛，这我承认，毕竟刚开始做的时候， 我只是想锻炼一下我的技术，不是想创造一个新的网站，但是我理解的创新是建立在基础功能之上，如果我没有实现这些最 基础的功能，那还叫社交网站吗，就像先出来的斗鱼TV，然后熊猫TV出来，大家没发现内容布局都一样吗，那不也是抄袭吗， 最基础功能大家都这么做，然后在此基础上添加自己的特色，所谓的“创新”。

1.1版本不是结束，路，才刚刚开始。。。
